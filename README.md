# Web-Scrapping-NPO-info
This project was completed as my volunteer job with the Class Consulting Group. My goal is to extract NPO contact information from a website called the Great Non-Profit. 

## Language used
Python

## Packages used
requests, pandas, bs4, urllib.parse

## Files in this project
Web Scrapping NPO information code.ipynb: The entire code I wrote to web scrape information
Nonprofit Org Contact Info Fixed.xlsx: The result excel file saved with extracted NPO contact information
